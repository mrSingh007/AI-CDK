# Spinner Component

## Purpose
Provide a compact inline loading indicator for short asynchronous states.

## What It Should Do
- Display a circular spinner for active loading states.
- Support size variants for use in buttons, inline text, and compact layouts.
- Remain lightweight and easy to embed in multiple components.

## Customization Expectations
- Allow color, track, stroke, duration, and size tuning through style tokens.
- Support consistent visual behavior across themes.

## AI Chatbot Usage
- Indicate short in-progress actions like sending prompts or fetching quick metadata.
- Show inline loading in chat controls and assistant action areas.

## Non-Goals
- Not a full-page loading layout.
- Not a replacement for skeleton placeholders in large content regions.
