# @ai-cdk/ag-ui

`@ai-cdk/ag-ui` is an Angular wrapper around AG-UI-compatible agent transports.

It manages streaming message state, tool-call driven UI payloads, and pending human-in-the-loop interactions so you can pair AG-UI streams with `@ai-cdk/ui/Renderer`.

Check [Storybook](https://mrsingh007.github.io/AI-CDK/) for all components and changes.
Contribute to project [HERE](https://github.com/mrSingh007/AI-CDK/)

## Installation

```bash
npm install @ai-cdk/ag-ui
```
